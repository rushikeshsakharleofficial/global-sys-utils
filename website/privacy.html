Privacy Policy

Last updated: June 2026

1. Overview

This privacy policy describes how the global-sys-utils project website (https://rushikeshsakharleofficial.github.io/global-sys-utils) collects, uses, and protects information. The website is a static informational site hosted on GitHub Pages. It does not run any server-side code and does not collect personal data beyond what is described below.

2. Information We Collect

2.1 GitHub Pages Hosting

This site is hosted on GitHub Pages. GitHub may collect standard web server access logs (IP addresses, browser user agents, page paths, timestamps) as part of their hosting infrastructure. Please refer to the GitHub Privacy Statement for details on how GitHub handles this data.

2.2 Contact Form

The contact form on the Contact page is a static form that does not submit data to any server in the current implementation. Responses are displayed in-browser only and no data leaves your device.

2.3 Local Storage

This website stores a single key (theme) in your browser's localStorage to remember your preferred color theme (dark or light). This data never leaves your device and is not transmitted to any server.

3. Cookies

This website does not set any first-party cookies. GitHub Pages may set cookies as part of their CDN or DDoS-protection infrastructure; these are governed by GitHub's privacy policy.

4. Third-Party Services

The website loads fonts from Google Fonts (Inter and JetBrains Mono). When your browser requests these fonts, Google may log your IP address and the referring URL. You can refer to the Google Privacy Policy for more details.

5. The global-sys-utils Software

The global-sys-utils software (the CLI tool itself) is installed and run locally on your own infrastructure. It does not send telemetry, usage data, or personal information to any external service. All log file data stays on your machines and your configured cloud storage (S3/GCS), which you control entirely.

6. Data Retention

Since we do not collect personal data on this website, there is no data retention policy applicable. Any access logs held by GitHub are subject to GitHub's own retention schedule.

7. Your Rights

If you are a resident of the EU/EEA or other jurisdictions with applicable data protection laws, you have rights including access to and deletion of your personal data. Since we do not collect personal data on this site, there is no data held by us to access or delete. For GitHub-held data, please contact GitHub directly.

8. Contact

If you have questions about this privacy policy, contact us at ramsharath@instantly.ai or open an issue on GitHub.

9. Changes to This Policy

We may update this policy from time to time. Changes will be reflected by updating the "Last updated" date above. Continued use of the site after any changes constitutes acceptance of the updated policy.

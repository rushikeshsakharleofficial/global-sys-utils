Releases & Downloads

Pre-built packages for Debian/Ubuntu (.deb) and Red Hat/Fedora (.rpm), for both amd64 and arm64 architectures.

Quick install

wget https://github.com/rushikeshsakharleofficial/global-sys-utils/releases/download/v2.2.0/global-logrotate_2.2.0-1_amd64.deb sudo dpkg -i global-logrotate_2.2.0-1_amd64.deb sudo systemctl enable --now global-sys-utils.service
wget https://github.com/rushikeshsakharleofficial/global-sys-utils/releases/download/v2.2.0/global-logrotate_2.2.0-1_arm64.deb sudo dpkg -i global-logrotate_2.2.0-1_arm64.deb sudo systemctl enable --now global-sys-utils.service
wget https://github.com/rushikeshsakharleofficial/global-sys-utils/releases/download/v2.2.0/global-logrotate-2.2.0-1.x86_64.rpm sudo rpm -ivh global-logrotate-2.2.0-1.x86_64.rpm sudo systemctl enable --now global-sys-utils.service
wget https://github.com/rushikeshsakharleofficial/global-sys-utils/releases/download/v2.2.0/global-logrotate-2.2.0-1.aarch64.rpm sudo rpm -ivh global-logrotate-2.2.0-1.aarch64.rpm sudo systemctl enable --now global-sys-utils.service
# Requires Go 1.21+ and Python 3.9+ git clone https://github.com/rushikeshsakharleofficial/global-sys-utils cd global-sys-utils make build sudo make install

All releases

System requirements

🐧
Linux kernel
3.10+ (glibc 2.17+). Tested on Ubuntu 20.04+, Debian 11+, RHEL 8+, Fedora 36+.
🐍
Python 3.9+
Required for cloud upload modules (boto3, google-cloud-storage). Installed automatically by .deb/.rpm post-install scripts.
🔧
systemd 245+
For daemon and timer units. If your distro uses SysVinit, you can run global-logrotate as a standalone cron job.
